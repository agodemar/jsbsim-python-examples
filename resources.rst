Related projects
----------------

* `JSBSim <https://github.com/JSBSim-Team/jsbsim>`_ – Open-source, multi-platform flight
  dynamics model (FDM) written in C++ with a Python wrapper.

* `JSBSim documentation <https://jsbsim-team.github.io/jsbsim-reference-manual/>`_ –
  Reference manual including the Python API.

* `PathSim <https://github.com/pathsim/pathsim>`_ – Block-based, time-domain system
  simulation framework in Python.

* `PathSim documentation <https://docs.pathsim.org>`_ – Tutorials, API reference, and
  examples.

* `PathView <https://view.pathsim.org>`_ – Browser-based graphical editor for PathSim
  block diagrams.  Export diagrams as Python scripts.

* `pathsim-flight <https://github.com/pathsim/pathsim-flight>`_ – Flight-dynamics blocks
  for PathSim, including a JSBSim wrapper and the International Standard Atmosphere.
